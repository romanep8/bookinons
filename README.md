# Site vitrine – Bookinons

Ce projet a été réalisé dans le cadre d'un projet de partiel en mars 2025.
Il est présenté ici à des fins de démonstration technique.
Sujet de partiel : Endosser le rôle d'un jeune entrepreneur qui lance sa start-up basée sur un concept original.

## Technologies utilisées
- HTML
- CSS (Flexbox / Grid)
- JavaScript

## Objectifs du projet
- Site vitrine responsive et accessible
- Structure claire
- UX/UI adapté à la cible
